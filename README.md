Ball Game

Petit jeu réalisé avec Unity dans lequel le joueur doit collecter tous les objets du niveau tout en évitant un ennemi qui le poursuit.

Technologies utilisées
-Unity 6
-C#
-Unity NavMesh

Ouvrir le projet
1-Cloner le dépôt.
2-Ouvrir le projet avec Unity 6.
3-Attendre la reconstruction du dossier Library.
4-Ouvrir la scène principale.
5-Lancer le jeu avec le bouton Play.

Ou

1-Télécharger le build à l'aide du release
2-Extraire les fichiers du .zip
3-Lancer le jeu à l'aide d'Unity
